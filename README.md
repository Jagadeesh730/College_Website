# SRM University of Science & Technology - College Website

A complete, modern, professional, premium-quality College Website built using only **HTML5**, **CSS3**, and **Vanilla JavaScript**.

## Features

- Fully responsive design (Desktop, Laptop, Tablet, Mobile)
- Sticky navigation bar with dropdown menus
- Hero image slider with auto-play and controls
- Animated statistics counters
- Scroll reveal animations (fadeIn, zoomIn, slideUp, slideLeft, slideRight)
- Dark mode with local storage persistence
- Gallery lightbox with keyboard navigation
- Faculty/department/course search with live filtering
- Admission form with client-side validation
- Contact form with draft auto-save to local storage
- Event countdown timers
- FAQ accordion
- Testimonials auto-slider
- Toast notifications
- Back to top button
- Loading screen with progress bar
- Google Fonts (Poppins, Montserrat)
- Font Awesome icons
- CSS custom properties (variables)
- CSS Grid & Flexbox layouts
- Glassmorphism effects
- Smooth transitions and animations

## Pages

1. **index.html** - Home page with hero slider, stats, about, departments, courses, facilities, placements, news, achievements, testimonials, FAQ
2. **about.html** - College history timeline, vision/mission, messages, accreditations
3. **departments.html** - Department cards with search functionality
4. **courses.html** - UG and PG course listings with details
5. **admissions.html** - Admission process, eligibility, fee structure, scholarships, application form
6. **placements.html** - Placement statistics, recruiters, process, internships
7. **faculty.html** - Faculty profiles with search by name/department
8. **events.html** - Upcoming and past events with countdown timers
9. **gallery.html** - Image gallery with lightbox
10. **contact.html** - Contact info, form, Google Maps embed

## Technologies Used

- HTML5 (Semantic markup)
- CSS3 (Custom properties, Grid, Flexbox, Animations, Transitions)
- Vanilla JavaScript (ES6+)
- Font Awesome 6 (Icons)
- Google Fonts (Poppins, Montserrat)

## How to Run

1. Open `index.html` in any modern web browser.
2. Navigate through pages using the navigation bar.
3. All features work without any server or build tools.
